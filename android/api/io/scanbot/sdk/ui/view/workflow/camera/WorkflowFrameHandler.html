Class WorkflowFrameHandler

  • All Implemented Interfaces:
    io.scanbot.sdk.camera.FrameHandler 
    
public class WorkflowFrameHandler
 implements FrameHandler

    Performs contour detection on camera preview frames. Outputs results via WorkflowFrameHandler.ResultHandler (.addResultHandler).

    Preview frame considered intercepted if at least one WorkflowFrameHandler.ResultHandler returns true from WorkflowFrameHandler.ResultHandler.handle
