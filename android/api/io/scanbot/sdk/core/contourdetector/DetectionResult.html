        DetectionResult(DetectionStatus status, List<Point> polygon, List<PointF> polygonF, Double detectionScore, List<Line2D> horizontalLines, List<Line2D> verticalLines)
        Parameters:
        status - Result status of contour detection
        polygon - detected polygon (if any) as absolute pixel coordinates.
        polygonF - detected polygon (if any) as floating point coordinates in [0.f, 1.f].
        detectionScore - detection score (if any) as double value in [0.0, 100.0].
        horizontalLines - Detected horizontal lines (can be used to snap while editing)
        verticalLines - Detected vertical lines (can be used to snap while editing)
