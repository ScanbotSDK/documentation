(nullable SBSDKMachineReadableCode *) - parseMetadata:
 

Detailed Description

The protocol any machine readable code parser must conform to. Each parser converts a SBSDKMachineReadableCodeMetadata into a subclass of SBSDKMachineReadableCode or nil.

Method Documentation

§ parseMetadata:()

- (nullable SBSDKMachineReadableCode *) parseMetadata: (nonnull SBSDKMachineReadableCodeMetadata *)  metadata

Parses machine readable code metadata and, on success, returns an instance of a SBSDKMachineReadableCode subclass.

Returns
The machine readable code subclass the receiver extracted from the MRC metadata or nil, if parsing failed.
