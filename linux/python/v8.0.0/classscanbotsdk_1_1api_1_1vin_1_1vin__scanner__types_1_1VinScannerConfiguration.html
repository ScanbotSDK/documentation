Parameters
extract_vin_from_barcodeIf true, the VIN scanner will also extract VINs from barcodes. Requires a license that allows barcode scanning in addition to VIN scanning. Default is false
ocr_resolution_limitMaximum image size (height or width) for OCR process. 0 - do not rescale. Default is 0
maximum_number_of_accumulated_framesMaximum number of accumulated frames to inspect before actual result is returned. Default is 3
minimum_number_of_required_frames_with_equal_scanning_resultMinimum number of accumulated frames that have equal result. Default is 2
