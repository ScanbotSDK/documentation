Parameters
result_accumulation_configConfiguration for how to accumulate results.
field_exclude_listNormalized names of the fields to exclude from the result.
configurationsList of configuration elements for the document data extractor.
return_cropsIf true, crops of the detected documents and fields will be returned in the result. Default is false
document_trust_modeLevel of trust in the authenticity of the extracted document. Default is TRUSTED
processing_modeScanners typically can produce better results from a single image if they are allowed to spend a longer time analyzing it. On the other hand, scanners can produce even better results if they can analyze multiple images of the same subject and cross-check and combine the results. The processing mode tells the scanner whether to optimize for single or multiple images of a subject. Default is AUTO
