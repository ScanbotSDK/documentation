Member Data Documentation

◆ EXACT

int EXACT = 0
static

Require seeing the same result multiple times to confirm it.

E.g. if the seen values are "John B. Doe", "John B.", "B. Doe", "John B. Doe" then the confirmation count for the value "John B. Doe" is 2 and for the rest it's 1. A field's value is considered CONFIRMED if its confirmation count is greater or equal to minConfirmations.

◆ INTERPOLATE

int INTERPOLATE = 1
static

Interpolate between seen values to generate the most likely true value.

E.g. if the seen values are "John X. Doe", "John B.", "B. Doe" then the most likely true value can be interpolated to be "John B. Doe". The interpolated value is considered CONFIRMED if each character in it has been seen at least minConfirmations times.
