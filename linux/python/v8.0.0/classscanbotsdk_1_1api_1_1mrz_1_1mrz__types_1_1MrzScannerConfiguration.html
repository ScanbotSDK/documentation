Parameters
frame_accumulation_configurationConfigure the frame accumulation process.
enable_detectionEnable MRZ detection. If disabled, the scanner skips the detection step and assumes that the input image is a crop of the MRZ area. Default is true
incomplete_result_handlingDefines how to handle incomplete MRZ results (e.g. caused by failed validation). Default is REJECT
return_cropsIf true, crops of the detected MRZ and its fields will be returned in the result. Default is false
