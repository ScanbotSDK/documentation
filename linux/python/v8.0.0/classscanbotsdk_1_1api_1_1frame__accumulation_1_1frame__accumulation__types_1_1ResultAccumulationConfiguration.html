Parameters
confirmation_methodMethod used to confirm a result. Default is EXACT
min_confirmationsNumber of confirmations required to consider a result confirmed (see ConfirmationMethod). Requiring more confirmations will increase the reliability of the result but also the time to gather enough confirmations. Default is 3
min_confidence_for_stable_fieldMinimum confidence required to consider a field confirmed. Default is 0.8
auto_clear_thresholdWill auto-clear the cache if this number of frames have been a different document type or empty. Default is 4
