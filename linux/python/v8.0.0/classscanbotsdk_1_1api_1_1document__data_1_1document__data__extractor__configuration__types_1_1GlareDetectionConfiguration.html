Detailed Description

Glare detection configuration.

The recognizer will detect whether fields are obstructed by glare. This feature is enabled by default. This configuration has no effect in SINGLE_SHOT mode.

Parameters
glare_detection_thresholdGlare detection threshold. The value should be in the range [0.0, 1.0]. Determines how much glare is acceptable relative to the text size. So 0.5 means that the glare can be up to 50% of the text size. If set to 0.0, the feature is disabled. Default is 0.5
