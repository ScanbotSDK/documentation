      • setGs1DecodingEnabled

         abstract Unit setGs1DecodingEnabled(Boolean gs1DecodingEnabled)

        When this set to true, the scanner assumes that the barcode can be a GS1 barcode. Turn it off, if you don't want to see decoded FNC1 characters ("]C1" and ASCII char 29). The default is true. NOTE: Currently works for CODE128 barcodes only!
