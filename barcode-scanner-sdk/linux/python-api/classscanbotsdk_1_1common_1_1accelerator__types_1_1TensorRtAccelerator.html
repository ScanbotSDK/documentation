Parameters
engine_pathDirectory in which the TensorRT engine files are stored. The SDK automatically generates the files if they are absent. Note, that it might take a few minutes to generate the engine files, therefore it is recommended to set the path to a directory that is not deleted between app launches. If the path is empty, the SDK will generate the engine files in the current working directory. Default is ""
