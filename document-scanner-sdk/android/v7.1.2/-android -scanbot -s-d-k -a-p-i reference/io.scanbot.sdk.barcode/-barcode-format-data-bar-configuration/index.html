data class BarcodeFormatDataBarConfiguration(var regexFilter: String = "", var minimumSizeScore: Double = 0.0, var addAdditionalQuietZone: Boolean = false, var minimum1DQuietZoneSize: Int = 6, var minimumNumberOfRequiredFramesWithEqualRecognitionResult: Int = 1, var gs1Handling: Gs1Handling = Gs1Handling.PARSE) : BarcodeFormatOneDConfigurationBase, Parcelable

minimum1DQuietZoneSize
Link copied to clipboard
open override var minimum1DQuietZoneSize: Int

Minimum quiet zone size on the left and right sides of a 1D barcode, measured in number of modules. The quiet zone is only for 1D barcodes with weak start/stop patterns, i.e. Iata2Of5, Industrial2Of5, Code25, Itf, Codabar, MsiPlessey and Code11.

minimumNumberOfRequiredFramesWithEqualRecognitionResult
Link copied to clipboard
open override var minimumNumberOfRequiredFramesWithEqualRecognitionResult: Int

Minimum number of frames on which a linear (1D) barcode has to be detected to be considered valid (live scanning only). Linear barcodes are more prone to false positives. Increasing the number of frames on which a barcode has to decode to the same content reduces the probability of a false positive.

minimumSizeScore
Link copied to clipboard
open override var minimumSizeScore: Double

Minimum acceptable value of a result BarcodeItem's sizeScore (between 0 and 1). Barcodes with a sizeScore less than this value will not be scanned. When set to 0, barcodes are returned no matter what their size is.

regexFilter

open override var regexFilter: String

Regular expression filter for barcode text. If the barcode text does not match the regular expression, it will not be scanned. The default is an empty string (setting is turned off).
