        Parameters

        • params: { captureFileUri?: string; imageFileUri: string; refreshOnEachFrame?: boolean }

            The file uri of the image to be used as a mock captured frame. If not provided, the imageFileUri will be used.

          • imageFileUri: string

            The file uri of the image to be used as a mock camera frame.

          • OptionalrefreshOnEachFrame?: boolean

            If true, the mock camera will refresh the frame on each request, simulating a live camera feed.
