SBSDK_API scanbotsdk_error_code_t scanbotsdk_date_validation_configuration_create ( int min_year,
int max_year,
const char * field_type_name,
scanbotsdk_date_validation_configuration_t ** result )

Creates a new instance of scanbotsdk_date_validation_configuration with given params and stores it in the location specified by the last argument.

As a rule if the function returns SCANBOTSDK_OK then it takes ownership of the pointers to scanbotsdk structures scanbotsdk_..._t* which otherwise would have to be freed by the caller by calling appropriate scanbotsdk_..._free and possibly invalidates them. The only exception is scanbotsdk_image_t* (if applicable) which is a long-lived object and should be freed by the caller. If the function returns an error code then the pointers to scanbotsdk structures remain valid and the caller is responsible for freeing them.

Parameters
min_yearMinimum year that is considered valid. Default is 0
max_yearMaximum year that is considered valid. Default is 2999
field_type_nameName, FullName, or NormalizedName of field type for which the year bounds should be applied. The value must not be null.
resultPointer to a variable that will store the created instance of scanbotsdk_date_validation_configuration_t
Returns
: Error code, SCANBOTSDK_OK if no errors were encountered
