The recognition mode

Enumerator
SCANBOTSDK_GENERIC_DOCUMENT_RECOGNITION_MODE_LIVE 

The recognizer will attempt to accumulate results over multiple frames.

SCANBOTSDK_GENERIC_DOCUMENT_RECOGNITION_MODE_SINGLE_SHOT 

The recognizer will only use the current frame and will spend additional time to ensure the best possible result.

enum scanbotsdk_generic_document_recognition_status_t

scanbotsdk_generic_document_recognition_status_t enum

The status of the recognition process

Enumerator
SCANBOTSDK_GENERIC_DOCUMENT_RECOGNITION_STATUS_SUCCESS 

The document was recognized successfully

SCANBOTSDK_GENERIC_DOCUMENT_RECOGNITION_STATUS_ERROR_NOTHING_FOUND 

No document was detected

SCANBOTSDK_GENERIC_DOCUMENT_RECOGNITION_STATUS_ERROR_BAD_CROP 

A document was detected, but it was at an angle/distance that was too large

SCANBOTSDK_GENERIC_DOCUMENT_RECOGNITION_STATUS_ERROR_UNKNOWN_DOCUMENT 

A document was detected, but it was not recognized as a supported document

SCANBOTSDK_GENERIC_DOCUMENT_RECOGNITION_STATUS_ERROR_UNACCEPTABLE_DOCUMENT 

A document was detected as a supported document, but it was not part of the accepted documents

SCANBOTSDK_GENERIC_DOCUMENT_RECOGNITION_STATUS_INCOMPLETE_VALIDATION 

All fields were recognized, but some of them failed validation
